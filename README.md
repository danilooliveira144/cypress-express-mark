# cypress-express-mark

Projeto de automação de testes de ponta a ponta para a aplicação Mark L

---

## Tecnologias e Versões

*   **Node.js:** `v18.12.1`
*   **Cypress:** `v12.2.0`

---

## Como iniciar o projeto localmente

Siga os passos abaixo para rodar o projeto localmente.

### pré-requisitos
Você precisa ter o **Node.js** e um gerenciador de pacotes (npm ou yarn) instalados na sua máquina.

### 1. Clonar o Repositório
````
git clone https://github.com/danilooliveira144/cypress-express-mark.git
````
### 2. Instalar as dependências e bibliotecas do Node.js
````
npm install
````
### 3. Instalar o cypress na versão 12.2.0
````
npm i cypress@12.2.0 --save-dev
````
### 4. Iniciar o cypress
````
npx cypress open
````
